    Version Control Software
    Git

    Technology changes with time flaws are removed and new features are added something with new update bugs may be generated. Git helps us to roll back to our previous state. Git works by making repositories and takings snap of changes we make and kept in a folder

    Nomenclature used in Git
    • Repositories ➤ It can contain folder ,files ,images ,videos ,spreadsheet and data sets.
    • Branch ➤Copy of same repository used to modify without disturbing the main code or repository.
    • Working Directory ➤Current working area where you write your code
    • Staging Area ➤Area where your code ,files and folder exist and there snap is taken and stored.
    • Git repository ➤Area where your repository and changes made are stored
